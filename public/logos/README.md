# Company logos for the "Trusted by" carousel

Place your logo image files in this folder (`public/logos/`).

## File names

Use these **exact filenames** so the site can display them:

| Company           | Filename            |
|-------------------|---------------------|
| BOSCH             | `bosch.png`         |
| NESTLE            | `nestle.png`        |
| CHEP              | `chep.png`          |
| Daniels Health    | `daniels-health.png`|
| Thales            | `thales.png`        |
| Asahi             | `asahi.png`         |
| Dulux             | `dulux.png`         |
| Saputo            | `saputo.png`        |
| Symbion           | `symbion.png`       |
| Textor            | `textor.png`        |
| Tip Top Bakeries  | `tip-top-bakeries.png` |

## Format

- **Format:** PNG or SVG (transparent background works best on the teal bar).
- **Size:** Around 120–200px wide is enough; the carousel will scale them.
- **Style:** Prefer white or light logos so they stand out on the teal background.

If a logo file is missing, the company name will show as text instead.
