# Australian Machine Vision Website

Production-ready B2B Computer Vision AI company website built with Next.js (App Router), Tailwind CSS, and shadcn/ui.

## Tech stack

- **Next.js** (App Router)
- **Tailwind CSS** with custom theme
- **shadcn/ui** (Button, Card)
- **Framer Motion** (installed for future use)
- **TypeScript**
- **Lucide React** (icons)

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` – Development server
- `npm run build` – Production build
- `npm run start` – Run production server
- `npm run lint` – Run ESLint

## Project structure (Day 1)

- `app/` – Routes and layout (Home, Customer Stories, About, Contact)
- `components/layout/` – Navbar, Footer
- `components/ui/` – Button, Card
- `lib/utils.ts` – `cn()` and utilities
- Theme: off-white background, teal/emerald accent, soft shadows, glass utilities
